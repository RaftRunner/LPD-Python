What are functions?

A function is a block of code which only runs when it is called.
Creating Functions

In Python a function is defined using the def keyword:

def myFirstFunction():
. print("Hello there!")
#This will not print anything because we now need to call the function in order to execute it

Calling functions:

  • By calling a function we are executing its code
  • We simply need to type the functions name followed by "()"

def myFirstFunction():
. print("Hello there!")

myFirstFunction()
#This will print "Hello there!" as it has now been executed

Arguments and how to use them

Information can be placed into functions as arguments.
What are arguments?

Arguments are specified inside the (). You can add as many arguments as you want, just separate them with a comma.

def myFirstFunction(firstName):
. print(firstName + "Smith")

myFirstFunction("James")
myFirstFunction("Ricky")
#This will print James Smith and Ricky Smith
#When we call the function we can enter a name and it will output that name plus what we wrote in the function

Number of Arguments:

  • A function must be called with the correct number of arguments.
  • If a function has 2 arguments, you must call the function with 2 arguments used.
  • The number of arguments must match.

def myFirstFunction(firstName, lastName):
. print(firstName + " " + lastName)
#The " " allows us to put a space between the names, otherwise they print as one word

myFirstFunction("James", "Smith")
myFirstFunction("Donald", "Jackson")
#They will print "James Smith" and "Donald Jackson" as it has now been called

Arbitrary Arguments:

  • If you don't know how many arguments there will be, then put (*x).
  • x represents a random parameter name here.

def myFirstFunction(*cars):
. print("My favourite car company is " + cars[2])

myFirstFunction("Ford", "Volkswagen", "Toyota")
#This will print Toyota as the favourite car company as it is at position of Index 2

Keyword Arguments:

  • You can also use arguments with the key = value syntax.
  • This way the order of the arguments does not matter.

def myFirstFunction(dog1, dog2, dog3):
. print("The best dog breed is " + dog2)

myFirstFunction(dog1 ="Patterdale", dog2 = "Greyhound", dog3 = "Pitbull")
#This prints "Greyhound" as the best dog breed
