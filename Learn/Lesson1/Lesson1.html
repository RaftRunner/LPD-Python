How to make a Python program

Here are 3 common ways to create and run a Python program
1. We can write our Python code in the cmd.

>>> print("Hello World!")
Hello World!

2. We can create a python file somewhere, using the .py file extension, and run it in the Command Line by typing in python followed by the name of the file.py.

C:\Users\Tyler>python hello.py
Hello World!

3. We can download an IDE and execute our code there.

print("Hello World!")

What does print do?

The print() function will print a specific message, messages, or another standard output device. You can print anything as long as it works!

Printing 1 Object:

print(" ") means if there is a word, or multiple words in between the same set of speech marks then they will count as one object.

print("Hello World!")
Printing 2 Objects:

We can also use print(" ", " ") as this signifies two separate objects that will be printed.

print("Hello World!", "How are you?")
Printing Multiple Objects:

You're able to use print(" ", " ", " ") as many times as you want.

print("Hello!", "How are you?", "This print function has 3 objects now.")

Using Multiple Print Functions:

You're able to use print(" ") as many times as you want.

print("Hello!")
print("I like to learn!")
print("Python is fun!")

Printing Other Variables:

We will go into variables later on, but this shows that print can be used for other things too.

x = 5
print(x)
y = ("Red", "Yellow", "Green")
print(y)

What are comments and how do we use them?

Comments allow us to write things that the program will ignore.

What is the point of a comment?

Comments can be used to:

  • Explain code
  • Make the code easier to read and understand
  • Prevent certain code being executed
How to make a comment:

Single line comments are used simply by typing '#'.

# This is a comment
#So is this

We can also make comments that span multiple lines using this:

'''
Now this comment
Is multiple lines
Instead of just one
'''
Comments are always ignored:

The program will never acknowledge comments.

#Hello
print("Hello World!")

No matter where they are:

print("Hello World!")#Hello
Comments can also be used to stop code from being executed:

We can stop something from being executed in the program by commenting it out.

#print("Hey there!")
print("Hello World")
