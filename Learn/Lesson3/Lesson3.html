What are the data types?

Data types define the data that a variable will contain and actions it can perform.
What are the data types?

So far we have seen Text types:

1. String - Used to represent text rather than just numbers.

x = "Hello World!"
userName = "Number 1 Rick"

We have also seen Numeric Types:

1. Integer - A whole number that can be stored in a variable.

x = 3
y = 15

2. Float - A decimal number. It will always end in a decimal.

x = 3.0
y = 15.3

And lastly, Bool Types:

4. Boolean - Allows you determine whether something is true or false.

x = 3
print(x > 2) #Prints True
print(x == 3)#Prints True
print(x < 2)#Prints False


Now we will look at sequences, maps, and sets.
Sequence Types

It is a type that stores multiple items at a time. Types are lists, tuples, and range.

myList = ['Item1', 'Item2', 'Item3']

Sequence Types include lists, tuples, strings, and ranges.
Mapping Types

It is a dictionary that has a key, and a value, then maps them together so they're related.

personalInfo = {"name" : "Bob", "age" : 28}

A dictionary is a collection which is unordered (depending on your version of Python), changeable and indexed, so you can access specific parts of it and change it where necessary.
Set Types

Sets are used to store multiple items in a single variable and are used with curly brackets {}.

x = {"Apple", "Carrot", "Potato"}

A set is a collection which is unordered, unchangeable, and unindexed. We can add and remove items if we wish.

How to check the data type

You can get the data type of any object by using the type() function
Checking common data types

Int:

x = 3
print(type(x))
#This prints '< class 'int' >'

Float:

x = 3.0
print(type(x))
#This prints '< class 'float' >'

String:

x = ("Hello World")
print(type(x))
#This prints '< class 'str' >'

List:

x = ["Green", "Purple", "Blue"]
print(type(x))
#This prints '< class 'list' >'

Tuple:

x = ("Cat", "Dog", "Turtle")
print(type(x))
#This prints '< class 'tuple' >'

Dict:

x = {"name" : "Bob", "age" : 28}
print(type(x))
#This prints '< class 'dict' >'

Bool:

x = True
print(type(x))
#This prints '< class 'bool' >'
