What are loops?

Looping means repeating something until a particular condition is satisfied.
There are 2 types of loops

While Loops:

  • While loops can execute a set of statements as long as a condition is true.

x = 1
while x < 11:
. print(x)
. x += 1
#This will print all numbers from 1 to 10
#If x += 1 wasn't there then the loop would never end, it would keep printing 1

For loops:

  • For loops are used for iterating over a sequence.

dogs = ["Greyhounds", "Patterdales", "Pitbulls"]
for x in dogs:
. print(x)
#This will print all the items in the variable "dogs"

While loops

Here we will look at while loops in more depth
Break Statements

With the break statement we can stop the loop even if the while condition is true:

x = 1
while x < 11:
. print(x)
. if(x == 4)
. . break:
. x += 1
#This will only print numbers up to 4
#The break statement ensures the loop ends at 4

Continue Statements

Continue statements stop the current iteration, and continue with the next:

x = 3
while x < 8:
. print(x)
. if(x == 4)
. . continue:
. x += 1
#This will only print numbers from 3 to 8, but will skip 4

Else Statements

Else statements can be used to execute code when the condition is no longer true:

x = 1
while x < 7:
. print(x)
. x += 1
. . else:
. print("x is no longer less than 7")
#This will only print numbers from 1 to 5, and then the "x is no longer..." line

For loops

Here we will look at for loops in more depth
Break Statements

With the break statement we can stop the loop before it has looped through all the items:

dogs = ["Greyhounds", "Patterdales", "Pitbulls"]
for x in dogs:
. print(x)
. if(x == "Patterdales"):
. . break
. print(x)
#This will only print "Greyhounds" as the loop stops at "Patterdales"

Continue Statements

With the continue statement we can stop the current iteration of the loop, and continue with the next:

dogs = ["Greyhounds", "Patterdales", "Pitbulls"]
for x in dogs:
. print(x)
. if(x == "Patterdales"):
. . continue
. print(x)
#This will only print "Greyhounds" and "Pitbulls" as the loop skips "Patterdales"

Range Functions

To loop through a set of code a specified number of times, we can use range().

for x in range(5)
. print(x)
#This will print numbers starting from 0 up to 4
#We always start from 0

We can use the start parameter to choose a number to start from, and we can include a parameter to determine how much the loop goes up by for each number.

for x in range(2, 7)
. print(x)
#This will print numbers starting from 2 up to 6

for x in range(2, 11, 2)
. print(x)
#This will print 2, 4, 6, 8, 10 as it is incrementing by 2


Else Statements

Else in a for loop specifies a block of code that will be executed when the loop is finished:

for x in range(2, 7)
. print(x)
else:
. print("Loop has finished.")
#This will print 2 to 6 and then the "Loop has finished" line
