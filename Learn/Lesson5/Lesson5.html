What are if statements?

The if statement executes a block of code if a specified condition is true.
If Statements

Here we will see a basic example of an if statment:

x = 30
y = 15
if x > y:
. print("x is bigger than y")
#You must make sure to indent the line after the if statement
#This can be done by clicking the tab button when you are coding

Here is another example:

x = 55
y = 67
if x <= y:
. print("x is smaller or equal to y")
#Once again, remember to indent the print statement

One last example:

x = 20
y = 20
if x == y:
. print("x is equal to y")

Elif statements

The elif keyword is a way of executing a different condition if previous conditions were false.
Elif Statements

Here we will see a basic example of an elif statment:

x = 30
y = 30
if x > y:
. print("x is bigger than y")
elif x == y:
. print("x is equal to y")

Here is another example:

y = 30
if y > 15:
. print("y is bigger")
elif y < 15:
. print("y is less than 15")
elif y == 15:
. print("y is equal to 15")

Else statements

The if/else statement executes a block of code if a specified condition is true. If the condition is false, another block of code can be executed.
Else Statements

Here we will see a basic example of an else statment:

x = 1000
y = 250
if y > x:
. print("y is bigger than x")
elif x == y:
. print("x is equal to y")
else:
. print("x is bigger than y")

Here is another example:

y = 30
if y > 15:
. print("y is bigger")
elif y < 15:
. print("y is less than 15")
elif y == 15:
. print("y is equal to 15")
