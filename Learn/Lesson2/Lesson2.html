What is a variable?

A variable is a container for storing a value of data
What are the main variable types?

1. String - Used to represent text rather than just numbers.

x = "Hello World!"
userName = "Number 1 Rick"

2. Integer - A whole number that can be stored in a variable.

x = 3
y = 15

3. Float - A decimal number. It will always end in a decimal.

x = 3.0
y = 15.3

4. Boolean - Allows you determine whether something is true or false.

x = 3
print(x > 2) #Prints True
print(x == 3)#Prints True
print(x < 2)#Prints False

How to create variables

A variable is created when you assign it a value. We can call a variable anything as long as it follows the rules.
Variable names

Legal names:

myname = ("Ricky")
myName = ("Ricky")
Myname = ("Ricky")
MyName = ("Ricky")
my_name = ("Ricky")
_my_name = ("Ricky")
MYNAME = ("Ricky")
myname1 = ("Ricky")


Illegal names:

1myname = ("Ricky")
my-name = ("Ricky")
my name = ("Ricky")
Multiple Word Variable Names

To make variable names with more than one word readable, you can use the following techniques:

Camel Case:

#Every word except the first is capitalised
myFirstName = ("Ricky")

Pascal Case:

#Every word is capitalised
MyFirstName = ("Ricky")

Snake Case:

#Every word is seperated by an underscore
my_first_name = ("Ricky")
Overriding a variable

Variables can even change type after they have been set.

x = 5 #Here x is of type int
x = "I like apples." #Now it is of type str

How to use a variable

To use a variable you must declare it and assign a value for which it will be used for.

How can variables be used?

Variables can be used to:

  • Store information
  • Be referenced and used in a program
  • Label data with useful names, so programs can be better understood by everyone
One value can have multiple variables

And you can assign the same value to multiple variables in one line:

x, y, z = "Yes"
Outputting a variable

You can output a variable using the print function:

x = 5
print(x) #This will print 5

myFirstName = "Ricky"
myLastName = "Johnson"
print("My name is: " + myFirstName + " " + myLastName)
#This will print 'My name is: Ricky Johnson'
